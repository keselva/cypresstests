# cypresstests

## This is a sample project to run cypress tests in firefox and chrome with percy.io in shippable

# Requirements
#### node 10 or above
#### cypress 4.5.0
#### firefox 76.0

# Execution Steps
#### npm install
#### npm run cypress:percy

# CI pipeline
#### Tests are integrated in Shippable CI
